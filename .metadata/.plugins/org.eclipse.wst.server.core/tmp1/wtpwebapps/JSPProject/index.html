경로 지정하기 최상위 주소 경로 (프로젝트 명) == 최상위 파일경로(webapp폴더) 1) 주소로 파일을 요청하는 경우 - webapp폴더를 기준으로 하위로 접근하는 형식 (/webapp/forResult.jsp) ex)/JSPProject/forResult.jsp 2)주소로 Servlet을 요청하는 경우 - Servlet에 작성된 @WecServlet("주소")어노테이션의 주소는 실제하지 않는 가상의 주소 작성된 주소에 실제 파일 x , 요청이 오면 특정 서블릿이 처리 - webapp폴더 기준x 프로젝트 명(/JSPProject)을 기준으로 일치하는 Servlet을 조회 상대경로 : 현재 주소 또는 현재 파일을 기준으로 다른 주소나 경로를 나타내는 방법

요청주소경로 : 인터넷 주소창, form태그 action, a태그의 href등에 작성 되는 주소
파일경로 : 프로젝트 내의 js파일 위치, css위치 등
-java폴더 : *.java같은 소스코드 또는 코드관련 XML파일이 저장되는 폴더
-webapp폴더 : 실제로 웹상에 배포되는 코드가 저장되는폴더 (html, css, jsp, image + class파일,배포서술자(web.xml))
서블릿에서 jsp경로 지정시(절대경로만 가능) 제일 앞 "/"는 "webapp"폴더를 가르키는 기호 (webapp폴더가 배포 최상위 폴더)
